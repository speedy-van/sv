# 🚀 iOS Build - Quick Start

## ⚡ Super Quick Start (Copy & Paste)

```bash
cd mobile/expo-driver-app
npm install -g eas-cli
eas login
eas build:configure
eas credentials
eas build --platform ios --profile production
eas submit --platform ios --latest
```
