Asset Placeholders

The following files are placeholders and should be replaced with actual images:

- icon.png (1024x1024) - App icon
- splash.png (1284x2778) - Splash screen
- adaptive-icon.png (1024x1024) - Android adaptive icon
- favicon.png (48x48) - Web favicon
- notification-icon.png (96x96) - Notification icon

For now, Expo will use default placeholders.

The app will work fine on Expo Go without custom assets!
