# Emotes Assets

This directory contains emote images for the driver reaction system.

## Current Emotes

- `celebrate.png` - Celebration emote for order acceptance
- `thumbs_up.png` - Thumbs up for route acceptance
- `facepalm.png` - Facepalm for order decline
- `confused.png` - Confused face for route decline
- `victory.png` - Victory pose for job completion
- `thinking.png` - Thinking face for job start
- `wave.png` - Waving goodbye for job cancellation

## Requirements

- File size: Under 200KB per emote
- Format: PNG preferred, Lottie JSON for animations
- Dimensions: 128x128px recommended
- Transparent background

## Configuration

Emote triggers and animations are configured in `emotes-config.json`.

## Usage

Emotes are automatically triggered based on driver actions and can be toggled in Settings.
